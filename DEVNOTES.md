# Build instructions

## Building

- Either run:

        pip install https://github.com/quantori/kifparser

- Or:

        git clone https://github.com/quantori/kifparser
        pip install ./kifparser

## Unit tests and linting

- To run unit tests and linting:

        tox

- In order to install tox:

        pip install tox
